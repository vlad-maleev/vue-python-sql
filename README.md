# vue-python-sql

## Project setup 
```
cd client

npm install
```

### Start Python server
```
cd server

python app.py
```


### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
